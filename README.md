
# Sistema de Login

Projeto em JavaScript desenvolvido para simular um sistema de login, utilizando um vetor de usuários com validação de email, senha e credenciais expiradas.

## Funcionalidades
- Validação de login
- Verificação de credenciais expiradas
- Tratamento de usuário inexistente
- Validação de senha incorreta

## Tecnologias
- JavaScript
- Git
- GitHub
